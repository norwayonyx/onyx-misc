🌙 Moonlight + Sunshine

// open source game streaming from your own PC

What is this?

Moonlight and Sunshine together let you stream games from your gaming PC to any other device — a laptop, a weaker PC, a phone, or a TV — over your local network or the internet. It's like GeForce NOW but self-hosted and completely free.

Cost
Free & Open Source
Host GPU
Any (AMD, Nvidia, Intel)
Latency
Very low on LAN
Quality
Up to 4K / 120fps
// Runs on the gaming PC (host)
☀️ Sunshine
The server software. Captures your screen and streams it. Replaces the old NVIDIA GameStream requirement — works on any GPU.
// Runs on the device you're playing on (client)
🌙 Moonlight
The client app. Connects to Sunshine, receives the stream, and sends your inputs back. Available on Windows, Mac, Linux, Android, iOS, and Raspberry Pi.

01 Install Sunshine (Host PC)

Install this on the powerful PC that will be running the games.

↗ Sunshine releases on GitHub
01
Download and run the Sunshine installer
02
Sunshine will open a web interface at https://localhost:47990 — open it in your browser
03
Create a username and password when prompted — this protects your Sunshine admin panel
04
Go to Configuration → General and set your preferred stream resolution and framerate
05
Make sure Sunshine is set to run on startup so it's always ready
// Tip: Sunshine runs a local HTTPS server. Your browser will warn about the certificate — this is expected, just click "proceed anyway".

02 Install Moonlight (Client Device)

Install this on whatever device you want to stream to — a laptop, phone, TV stick, etc.

↗ moonlight-stream.org
01
Download and install Moonlight for your platform
02
On the same local network, Moonlight will auto-detect your Sunshine host — it should appear on the home screen
03
Click the host to connect — you'll see a PIN code on screen
04
Enter the PIN in the Sunshine web interface under PIN to pair the devices
05
You're paired — click any app or game to start streaming

03 Streaming Over the Internet

To stream from outside your home network you need to expose your host PC. The easiest way, consistent with the rest of this site, is Cloudflare Tunnel — but for game streaming, direct port forwarding usually gives better latency.

// Option A — Port Forwarding (best performance)

01
Log into your router admin panel (usually 192.168.1.1)
02
Forward the following ports to your host PC's local IP:
PortProtocolPurpose
47984TCPHTTPS
47989TCPHTTP
48010TCPRTSP
47998–48000UDPVideo / Audio / Control
03
In Moonlight on the client, manually add your home's public IP address as a host

// Option B — Tailscale (easiest, no port forwarding)

Tailscale creates a private VPN between your devices with zero configuration. Install it on both the host and client, and they'll see each other as if on the same local network.

↗ tailscale.com — free for personal use
⚠️ Internet streaming note: Game streaming requires good bandwidth. Aim for at least 20 Mbps upload on the host and 20 Mbps download on the client for a smooth 1080p/60fps experience.

Troubleshooting

// Host not appearing in Moonlight

Make sure both devices are on the same network. Check that Sunshine is running — open https://localhost:47990 on the host to verify.

// High latency

Lower the bitrate and resolution in Moonlight settings. On WiFi, try moving closer to the router or switching to a 5GHz network.

// Black screen after connecting

In Sunshine settings, try switching the encoder (software vs hardware). Also make sure your GPU drivers are up to date.
