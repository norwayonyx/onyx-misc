Play with Friends

🎲 Browser Games with Friends

// no installs, no accounts needed — just send a link

These are the best free browser-based games and platforms for playing with friends instantly. Most need no account — just open a link and share the room code.

// Pro tip: Run these alongside a Discord voice call for the best experience. No extra setup needed.

01 Netgames.io — Social Deduction Games

netgames.io No account needed

Free browser implementations of popular social deduction card games. One person creates a room, shares the link, everyone joins instantly.

↗ netgames.io

// How to start a game

  1. Go to netgames.io
  2. Pick a game and click New Game
  3. Share the room link with friends
  4. Once everyone joins, hit Start

// Games available

5–10 players

02 BoardGameArena — Real Board Games

boardgamearena.com Free tier

Massive library of real licensed board games, fully playable in the browser. Free accounts can play most games — one player in the group needs a premium account to create tables for premium-only games, but most popular ones are free.

↗ boardgamearena.com

// How to start a game

  1. Create a free account at boardgamearena.com
  2. Search for a game in the library
  3. Click Play Now → Create a table
  4. Set the table to Private and share the link or invite friends directly
  5. Once everyone is seated, start the game

Other Quick Browser Games

// Drawing & Guessing

skribbl.ioNo account

Free browser Pictionary. One person draws, everyone else guesses. Create a private room and share the link.

↗ skribbl.io
Gartic PhoneNo account

The telephone drawing game — someone writes a prompt, the next person draws it, the next person describes the drawing, and so on. Hilarious results.

↗ garticphone.com

// Trivia & Quiz

KahootFree tier

Host a quiz and friends join with a game PIN on their phone. Huge library of free community quizzes on every topic imaginable.

↗ kahoot.com

// Geography

GeoGuessr (free mode)Partially free

Drop into a random Google Street View location and guess where in the world you are. The free tier lets you play the classic world map. Battle Royale mode is great with friends.

↗ geoguessr.com
