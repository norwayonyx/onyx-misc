What is Parsec?

Parsec streams your PC screen to a friend over the internet with very low latency — but more importantly, it lets friends connect their controllers and play local co-op games with you remotely. If a game supports two players on the same couch, Parsec makes it work online.

01 Install Parsec

Both the host and all friends need to install Parsec and create a free account.

↗ parsec.app/downloads

Create a free account at parsec.app — both host and guests need one.

02 Host: Share Your PC

01
Open Parsec and log in
02
Click Share in the top right — this makes your PC visible to friends
03
Share your Parsec username or send a share link to friends
04
Launch the game you want to play
05
When a friend connects you'll get a notification — accept it
// Tip: In Parsec settings under Host, set your resolution and framerate. 1080p / 60fps is a good starting point. Lower it if friends experience lag.

03 Guest: Connect to a Friend

01
Open Parsec and log in
02
Search for your friend's Parsec username under the Friends tab, or open their share link
03
Click Connect — their screen will appear in a window on your PC
04
Plug in a controller if the game supports it — Parsec will forward your inputs to the host

04 Controller Setup

This is where Parsec shines. When a guest connects with a controller plugged in, it shows up on the host PC as a real gamepad — games see it exactly like a second controller plugged into the same computer.

// Supported inputs

  • Xbox controllers (best compatibility)
  • PlayStation controllers (DualShock 4, DualSense)
  • Generic USB / Bluetooth gamepads
  • Keyboard & mouse (host can assign these as a virtual controller)
⚠️ Note: Parsec free tier allows up to 4 guests. The game itself must support local co-op — Parsec doesn't add online to games that don't have any multiplayer at all.

Good Games to Try

Any game with local co-op works. Some great ones that are free or cheap:

  • Stardew Valley — farming co-op, very relaxed
  • Terraria — 2D survival adventure
  • Cuphead — brutal but fun 2-player run & gun
  • Spelunky 2 — chaotic co-op platformer
  • River City Girls — beat em up
  • Any fighting game — Street Fighter, Mortal Kombat, etc.
  • Retro games via emulators — use Parsec with RetroArch for classic co-op

Troubleshooting

// High latency / lag

Lower the host's stream quality in Settings → Host. Try 720p / 30fps. The host's upload speed is the bottleneck.

// Controller not recognized

In Parsec settings on the guest side, make sure Virtual Gamepad Type is set to Xbox 360 for best compatibility.

// Black screen on connect

The host may need to enable GPU acceleration in Parsec host settings, or switch from integrated to dedicated GPU in the settings.
