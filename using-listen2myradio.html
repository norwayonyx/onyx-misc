// Quick Start — 5 Minutes
  1. Install VB-Cable
  2. Install BUTT
  3. Create a free server on Listen2MyRadio
  4. Enter the server details in BUTT
  5. Start streaming
  6. Share the stream URL

Pipeline Overview

Spotify / Music Player
↓
VB-Cable (routes audio)
↓
BUTT (broadcasts audio)
↓
Listen2MyRadio (hosts stream)
↓
Friends / Listeners

01 Install Required Software

// Spotify (optional)

Use Spotify or any other music player to play what you want to broadcast.

↗ spotify.com/download

// VB-Audio Virtual Cable

Routes audio from Spotify into the broadcasting software. After installing, restart your computer and set CABLE Input as your Spotify output device.

↗ vb-audio.com/Cable

// BUTT (Broadcast Using This Tool)

Broadcasting software that sends your audio to the radio server.

↗ danielnoethen.de/butt

02 Create a Free Radio Server

↗ listen2myradio.com

Create a Free Icecast server. You'll receive credentials like:

Hostname:        example.listen2myradio.com
Port:            12345
Mount:           /stream
Stream Password: ********

Save these — you'll need them in the next step.

03 Configure BUTT

Open BUTT and go to Settings → Server → Add. Fill in your server details:

FieldValue
TypeIcecast
Addressexample.listen2myradio.com
Port12345
Mountpoint/stream
Icecast Usersource
Password(your stream password)

Save the profile.

04 Configure Audio Settings

Go to Settings → Audio and use these recommended settings:

SettingValue
Audio DeviceCABLE Output (VB-Audio Virtual Cable)
CodecMP3
Bitrate96 kbps
Samplerate44100 Hz
ChannelsStereo
Bitrate ModeCBR

05 Start Streaming

Click Start Streaming in BUTT. You should see:

Connecting...
Connection established

Audio is now being sent to the Listen2MyRadio server.

06 Test & Share the Stream

Open the stream URL in a browser or VLC to verify it's working:

http://example.listen2myradio.com:12345/stream

In VLC: Media → Open Network Stream and paste the URL.

Then send that same URL to your friends. They can listen using a web browser, VLC, or any app that supports radio streams.

Troubleshooting

No Audio
Make sure Spotify (or your audio program) is outputting to CABLE Input.
Connection Failed
Double-check your Hostname, Port, Mountpoint, and Stream Password in BUTT settings.
Audio Stuttering
Lower the bitrate to 96 kbps in Settings → Audio.
