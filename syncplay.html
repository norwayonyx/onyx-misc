What is Syncplay?

Syncplay lets you and your friends watch the same video file at exactly the same time — when one person pauses, everyone pauses. When someone seeks, everyone seeks. You each need a copy of the video file, and Syncplay keeps your playback perfectly in sync.

Cost
Free & Open Source
Works on
Windows, Mac, Linux
Requires
Same video file on each PC
Players supported
VLC, MPV, MPC-HC

01 Install Syncplay

Download and install Syncplay on every computer that will be watching.

↗ syncplay.pl/download

Syncplay works with VLC, MPV, and MPC-HC. If you don't have a video player installed, grab VLC:

↗ videolan.org/vlc

02 First Time Setup

When you open Syncplay for the first time, a configuration window appears.

01
Set Server address to syncplay.pl and port 8995 — this is the free public server
02
Set a Username — this is how your friends will see you in the room
03
Set a Room name — make up any name, everyone joining needs to use the exact same one
04
Set the Media player path — point it to your VLC or MPV executable
05
Click Store configuration and connect
// Tip: The room name is like a password — anyone who knows it can join. Pick something unique to keep it just for your group.

03 Hosting a Watch Session

01
Make sure everyone has the same video file — exact same file works best. Syncplay matches by filename.
02
The host opens Syncplay and clicks Open file to load the video
03
Share the room name with friends so they can join
04
Friends open Syncplay, enter the same room name, and open their copy of the video
05
Once everyone is in the room, hit Play — Syncplay handles the rest

04 Using the Chat

Syncplay has a built-in chat window so you can react in real time without needing a separate app. It appears alongside the player controls. Combine it with a Discord voice call for the full experience.

Tips & Tricks

// Everyone pauses together

When any person in the room pauses, everyone else is automatically paused too. No more "wait wait wait, pause it!"

// Slow connection?

Syncplay has a slow sync feature — it gradually adjusts playback speed instead of jumping to catch up, which is much less jarring.

// Self-hosting the server

If you want a private server instead of using the public one, Syncplay is fully open source and the server component can be self-hosted. See the GitHub repo for instructions.
