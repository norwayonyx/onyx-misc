What is Excalidraw?

Excalidraw is a free, open source collaborative whiteboard that runs entirely in the browser. It has a hand-drawn aesthetic that makes diagrams feel casual and approachable. Share a link and multiple people can draw on the same canvas in real time.

Cost
Free & Open Source

Nothing — browser only
Account
Not required
Collaboration
Real-time, share a link

01 Starting a Collaborative Session

No setup required. Everything happens in the browser.

↗ excalidraw.com
  1. Go to excalidraw.com
  2. Click the Live collaboration button (top left, looks like two people)
  3. Click Start session
  4. Copy the link and send it to friends
  5. Everyone who opens the link can draw on the same canvas in real time
// Tip: The collaboration link is temporary — it exists as long as at least one person has the tab open. Save your work before closing!

02 Basic Tools

The toolbar is minimal but powerful. All tools have keyboard shortcuts:

  • V — Select / Move
  • R — Rectangle
  • E — Ellipse / Circle
  • A — Arrow
  • L — Line
  • T — Text
  • P — Pen / Freehand draw
  • H — Hand (pan around canvas)
  • Ctrl + Z — Undo
  • Ctrl + Shift + -/+ — Zoom out / in
// Tip: Double-click any shape to add text inside it. Double-click the background to quickly add a text element.

03 Saving Your Work

// Export as image

Go to Menu → Export image — saves as PNG or SVG. The PNG export has a transparent background by default.

// Save as .excalidraw file

Go to Menu → Save to disk — saves a .excalidraw file you can reload later with full editability.

// Share as read-only link

You can also export a shareable link that shows a read-only snapshot of your drawing — useful for sharing results without giving edit access.

What to Use It For

🗺️
Planning with friends
Map out a trip, plan a D&D dungeon, sketch out ideas together in real time
🎮
Game planning
Draw raid strategies, map out base layouts in survival games, plan routes
💡
Brainstorming
Mind maps, flowcharts, diagrams — the hand-drawn style keeps things low-pressure
🎨
Just drawing together
Collaborative art, doodling, pictionary-style drawing sessions

Self-Hosting

Excalidraw is fully open source and can be self-hosted if you want a permanent private instance. The public version at excalidraw.com is completely free with no limitations for most uses.
