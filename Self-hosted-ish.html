Audio Streaming

🖥️ Self-Hosted-ish Radio

// Icecast + BUTT + Cloudflare Tunnel

⚠️ Heads up: This method runs a server locally on your computer. It may be laggier than the Listen2MyRadio option, especially on slower connections. The stream also goes offline when you close the tunnel.

01 Install Icecast

Icecast is the server that distributes your audio stream to listeners. Download the Windows version.

↗ icecast.org/download

After installing, locate the Icecast installation folder — inside it you'll find icecast.xml.

02 Configure Icecast

Open icecast.xml in a text editor and make the following changes:

// Change Passwords

Find the authentication section and change the default passwords:

<authentication>
  <source-password>mypassword</source-password>
  <admin-password>adminpassword</admin-password>
</authentication>

// Verify Port

Make sure the port is set to 8000 (default, no change needed):

<port>8000</port>

// Optional: Improve Stability

This allows new listeners to buffer some audio and reduces stuttering:

<burst-size>65536</burst-size>

03 Start the Icecast Server

Inside the Icecast installation folder, run:

icecast.bat

A terminal window will open. Then go to http://localhost:8000 in your browser — you should see the Icecast Status Page.

04 Install BUTT

BUTT sends audio from your computer to the Icecast server.

↗ danielnoethen.de/butt

05 Configure BUTT

Open Settings and fill in the server details:

// Server Settings

FieldValue

Password(your source-password from icecast.xml)

// Audio Settings

06 Install VB-Cable Recommended

VB-Cable creates a virtual audio device that lets BUTT capture audio from Spotify or any other app.

↗ vb-audio.com/Cable

Download the ZIP, run VBCABLE_Setup_x64.exe, click Install Driver, and restart if prompted.

// Configure Windows Audio

01
Open Windows Sound Settings
02
Set Output device to CABLE Input (VB-Audio Virtual Cable) — this routes Spotify into the cable
03
In BUTT under Settings → Audio, set Input Device to CABLE Output (VB-Audio Virtual Cable)

07 Start Streaming

Click Start Streaming in BUTT, then visit http://localhost:8000 — you should see the /live mountpoint. Test it at:

http://localhost:8000/live

You should hear your audio.

08 Set Up Cloudflare Tunnel

Cloudflare Tunnel exposes your local server to the internet without port forwarding.

↗ Download cloudflared.exe
01
Rename the downloaded file to cloudflared.exe
02
Create a folder at C:\cloudflared and move the file there
03
Open Command Prompt and run: 
cd C:\cloudflared
cloudflared.exe tunnel --url http://localhost:8000
04
Cloudflare will generate a public URL like https://example-name.trycloudflare.com

Your stream URL will be:

https://example-name.trycloudflare.com/live

Share this with friends — they can open it in a browser, VLC, or any internet radio app.

Important Notes

⚠️ The tunnel must stay running. If you close the terminal running cloudflared, the public stream stops immediately.
