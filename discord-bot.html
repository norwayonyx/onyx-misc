01 Prerequisites

Python
  • Python 3.8 or higher — python.org
  • pip (comes with Python)
JavaScript
  • Node.js 16.9 or higher — nodejs.org
  • npm (comes with Node.js)

Both also require a Discord account and a text editor like VS Code.

02 Creating the Bot on Discord

This step is the same regardless of language.

01
Go to discord.com/developers/applications and log in.
02
Click "New Application", give it a name, and click "Create".
03
In the sidebar click "Bot" → "Add Bot" → confirm.
04
Under Token, click "Reset Token" then "Copy". Keep this secret!
05
Scroll to Privileged Gateway Intents and enable: Presence Intent, Server Members Intent, Message Content Intent.
⚠️ Warning: Never share your bot token or commit it to GitHub. If it's ever exposed, reset it immediately in the Developer Portal.

04 Writing the Bot Code

Python — bot.py
import discord
from discord.ext import commands
from dotenv import load_dotenv
import os

load_dotenv()
TOKEN = os.getenv('DISCORD_TOKEN')

intents = discord.Intents.default()
intents.message_content = True

bot = commands.Bot(command_prefix='!', intents=intents)

@bot.event
async def on_ready():
    print(f'✅ Logged in as {bot.user} (ID: {bot.user.id})')

bot.run(TOKEN)
JavaScript — index.js
require('dotenv').config();
const { Client, GatewayIntentBits } = require('discord.js');

const client = new Client({
    intents: [
        GatewayIntentBits.Guilds,
        GatewayIntentBits.GuildMessages,
        GatewayIntentBits.MessageContent,
    ]
});

client.once('ready', () => {
    console.log(`✅ Logged in as ${client.user.tag}`);
});

client.login(process.env.DISCORD_TOKEN);

05 Adding Commands

Python
@bot.command()
async def hello(ctx):
    await ctx.send(f'Hello, {ctx.author.name}! 👋')

@bot.command()
async def say(ctx, *, message: str):
    await ctx.send(message)

@bot.command()
async def info(ctx):
    embed = discord.Embed(
        title="Bot Info",
        description="I'm a custom Discord bot!",
        color=discord.Color.blue()
    )
    embed.add_field(name="Prefix", value="!", inline=True)
    embed.set_footer(text="Made with discord.py")
    await ctx.send(embed=embed)
JavaScript
client.on('messageCreate', message => {
    if (message.author.bot) return;

    if (message.content === '!hello') {
        message.reply(`Hello, ${message.author.username}! 👋`);
    }

    if (message.content.startsWith('!say ')) {
        const text = message.content.slice(5);
        message.channel.send(text);
    }

    if (message.content === '!info') {
        const { EmbedBuilder } = require('discord.js');
        const embed = new EmbedBuilder()
            .setTitle('Bot Info')
            .setDescription("I'm a custom Discord bot!")
            .setColor(0x0099FF)
            .addFields({ name: 'Prefix', value: '!', inline: true })
            .setFooter({ text: 'Made with discord.js' });
        message.channel.send({ embeds: [embed] });
    }
});

06 Inviting the Bot to Your Server

01
In the Developer Portal, go to your app → OAuth2 → URL Generator.
02
Under Scopes, check bot and applications.commands.
03
Under Bot Permissions, select: Read Messages, Send Messages, Embed Links, Read Message History.
04
Copy the generated URL, open it in your browser, select your server and click Authorize.

07 Running the Bot

Python
python bot.py
JavaScript
node index.js

You should see: ✅ Logged in as YourBot#1234

08 Keeping the Bot Online 24/7

Your bot stops when you close the terminal. Common solutions:

// Option A — PM2 on a VPS

npm install -g pm2

# Python bot
pm2 start bot.py --interpreter python3 --name my-bot

# JS bot
pm2 start index.js --name my-bot

pm2 save
pm2 startup

// Option B — Cloud Hosting (Railway / Render / Fly.io)

  1. Push your code to a private GitHub repo
  2. Sign up at railway.app, render.com, or fly.io
  3. Connect your repo and add DISCORD_TOKEN as an environment variable
  4. Deploy — the platform runs your bot automatically

// Option C — Replit (Easiest)

  1. Create a new Repl at replit.com (Python or Node.js)
  2. Add your token to Secrets (not in code!)
  3. Use UptimeRobot to ping your Repl URL every 5 minutes to keep it awake
